# Required Ingested Schema

| Field       | Description                                                                                                                                       |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| note_id     | A unique identifier for each entry in the dataset, used to join back to other structured data                                                     |
| timestamp   | When the note was created (ISO 8601 format)                                                                                                       |
| content     | Text content of the note                                                                                                                          |
| created_by  | Username of the record creator                                                                                                                    |
| entity_name | Name of the entity that the note refers to (e.g., company or person name)                                                                         |
| field_name  | _(Optional)_ Required only in case of multiple CRM fields with textual content. Name of the CRM field where the note is recorded                  |
| group_id    | _(Optional)_ Identifier linking related notes for the same entity. Use when multiple notes share a common context and should be analyzed together |
