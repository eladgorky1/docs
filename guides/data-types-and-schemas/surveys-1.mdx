# Required Ingested Schema

Each row represents a single response to a specific survey question submitted by a single respondent.

| Field       | Description                                                                                   |
| ----------- | --------------------------------------------------------------------------------------------- |
| joinable_id | A unique identifier for each entry in the dataset, used to join back to other structured data |
| timestamp   | When the survey was submitted (ISO 8601 format)                                               |
| content     | Text content of a single response                                                             |
| created_by  | Name of the survey submitter                                                                  |
| entity_name | Name of the related entity (e.g., customer name)                                              |
| field_name  | Relevant question in the survey                                                               |
