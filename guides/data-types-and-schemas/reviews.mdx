# Required Ingested Schema

Each row represents a single review.

| Field       | Description                                                                                    |
| ----------- | ---------------------------------------------------------------------------------------------- |
| joinable_id | A unique identifier for each review in the dataset, used to join back to other structured data |
| timestamp   | When the review was written (ISO 8601 format)                                                  |
| content     | Text content of the review                                                                     |
| created_by  | Name of the review writer                                                                      |
| entity_name | _(Optional)_ Name of the entity that the review refers to (e.g., company or product name)      |
| subject     | _(Optional)_ Subject of the review                                                             |
