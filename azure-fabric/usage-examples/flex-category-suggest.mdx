# Use FLEX_CATEGORY_SUGGEST with Azure Fabric

This page describes how to use the `FLEX_CATEGORY_SUGGEST` function in Azure Fabric to get category recommendations based on your data.

## Example

If you have at least 10 `"Other"` results, run:

```sql
%%sql
SELECT FLEX_CATEGORY_SUGGEST(
    'Did the customer report an issue? -> issue type -> [Billing, Technical]',
    'support_tickets'
);
```

**Sample output:**

```json
["Tablet", "Headphones", "Camera"]
```
