# Data Ingestion & Sharing

Flexor supports multiple methods for ingesting your enterprise data. Choose the approach that best fits your infrastructure, security requirements, and existing workflows.

## Cloud Storage (Bucket)

Upload files to your cloud storage bucket, and Flexor securely ingests them using credential-free authentication via Workload Identity Federation.

### Supported Object Storage

Connect Flexor to your existing cloud storage provider:

| Provider                                                                                                       | Description                     |
| -------------------------------------------------------------------------------------------------------------- | ------------------------------- |
| [Amazon S3](https://docs.flexorai.com/guides/data-ingestion-and-sharing/buckets/object-storage/amazon-s3)      | AWS Simple Storage Service      |
| [Azure Blob Storage](https://docs.flexorai.com/guides/data-ingestion-and-sharing/buckets/object-storage/azure) | Microsoft Azure Blob containers |
| [Google Cloud Storage](https://docs.flexorai.com/guides/data-ingestion-and-sharing/buckets/object-storage/gcs) | GCP Cloud Storage buckets       |

### File Types

Flexor processes two categories of files from your bucket:

| File Type                                                                                                    | Formats                                   | Use Case                                                |
| ------------------------------------------------------------------------------------------------------------ | ----------------------------------------- | ------------------------------------------------------- |
| [Tables](https://docs.flexorai.com/guides/data-ingestion-and-sharing/files-tables)                           | Parquet                                   | Structured data exports (CRM notes, chats, transcripts) |
| [Documents & Emails](https://docs.flexorai.com/guides/data-ingestion-and-sharing/files-documents-and-emails) | PDF, DOCX, EML, MSG, MBOX, PST, HTML, ZIP | Unstructured files and email archives                   |

## Data Warehouse Sharing

Share data directly from your data warehouse without file exports. Flexor connects to your DWH and processes data in place.

### Data Share

Native data sharing for real-time access without data movement:

| Platform                                                                                              | Description                   |
| ----------------------------------------------------------------------------------------------------- | ----------------------------- |
| [Snowflake](https://docs.flexorai.com/data-ingestion/data-ingestion-and-sharing/data-share/snowflake) | Native Snowflake Data Sharing |
