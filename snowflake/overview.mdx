Flexor integrates natively with Snowflake, enabling you to analyze all your unstructured data in the environment you already know, with the SQL syntax you're familiar with, and seamlessly connected to all your structured data.

## How It Works

Flexor runs directly inside your Snowflake environment. You query your data using familiar SQL syntax, and Flexor processes it in placeâ€”classifying text, extracting entities, and categorizing content at scale.

```sql
SELECT FLEX(message_id, 'Did the customer request a refund?') as flex
FROM support_tickets
```

No data pipelines to build. No external API calls to manage. Your data stays in Snowflake, and Flexor delivers structured insights alongside your existing tables.

## Available Functions

| Function                | Description                                                                 |
| ----------------------- | --------------------------------------------------------------------------- |
| `FLEX`                  | Run ad-hoc transformations on your unstructured data                        |
| `FLEX_CREATE`           | Create persistent columns that automatically process new data as it arrives |
| `FLEX_CATEGORY_SUGGEST` | Get suggestions for new categories based on your data                       |
| `FLEX_ARCHIVE`          | Archive and remove Flexor columns you no longer need                        |
