Flexor integrates natively with Databricks, enabling you to analyze all your unstructured data in the environment you already know, with the SQL syntax you're familiar with, and seamlessly connected to all your structured data.

## How It Works

Flexor runs directly inside your Databricks environment. You query your data using familiar SQL syntax.

## Available Functions

| Function                | Description                                                                 |
| ----------------------- | --------------------------------------------------------------------------- |
| `FLEX`                  | Run ad-hoc transformations on your unstructured data                        |
| `FLEX_CREATE`           | Create persistent columns that automatically process new data as it arrives |
| `FLEX_CATEGORY_SUGGEST` | Get suggestions for new categories based on your data                       |
| `FLEX_ARCHIVE`          | Archive and remove Flexor columns you no longer need                        |
