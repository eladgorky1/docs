# Databricks Installation Guide

This page describes how to install Flexor on Databricks.

## Prerequisites

- A user with `ACCOUNT ADMIN` permissions.
- Flexor to provide: `setup_secrets.py`, Flexor API key, and `.whl` file.

## Setup secrets

1. Run the following terminal commands on the admin's computer to install the necessary packages:

```bash
pip install databricks-cli
pip install databricks-sdk
```

2. Save the `setup_secrets.py` file (provided by your SA) to the admin's computer.
3. Execute the command below from the admin's terminal. Replace `<api_key>` with the appropriate value:

```bash
python setup_secrets.py \
    --scope flexor_secrets \
    --flexor-api-key <api_key>
```

## Grant access to users

Run the following command from the admin's terminal, replacing `<user_email>` with the actual user email address:

```bash
databricks secrets put-acl --scope flexor_secrets --principal <user_email> --permission READ
```

## Import the Flexor library

Import the Flexor library at the beginning of each notebook session:

```python
import flexor
flexor.init("flexor_secrets")
```

<Info>
  Run this initialization in the following scenarios:

  - First-time use in a new Databricks notebook.
  - After restarting the Databricks cluster.
  - When encountering `ModuleNotFoundError: No module named 'flexor'`.
</Info>
